# 飞机盒缺陷检测技能

## 运行方式

该 skill 由独立实时语音入口调用；用户不需要手动启动 `main.py`。

兼容直接启动检测任务：

```bash
python3 /home/elf/.picoclaw/workspace/fjh/skill/main.py --model 5319.rknn --target 1,2
```

## 实时语音流程

程序初始化结束后会全程实时监听，不再开启 5 秒录音任务。

1. 播报“请使用语音唤醒小智。”
2. 听到“小智”后，播报“我在。请说检测指令。”
3. 听到“检测”后，播报“已开启检测任务。请告诉我使用什么模型，例如一。”
4. 听到数字模型编号后，在当前飞机盒检测 skill 文件夹中查找对应 `.rknn` 模型；默认路径为 `/home/elf/.picoclaw/workspace/fjh/skill`，并播报“已使用×模型。”
5. 询问“请告诉我需要检测的缺陷目标标签：1污渍，2划痕，3缺角。”
6. 支持说“1”“衣”“污渍”“物资”“1污渍”“2”“划痕”“3”“缺角”等关键词或同音词选择目标标签；语音识别出“飞机河/飞鸡盒/飞几河”“屋子/吴字”“画很”“却认/雀任”等拼音相近词时，会按对应流程关键词处理。
7. 选择标签后播报“已使用×标签。”，并询问“请问是否还需要添加需要检测的缺陷目标标签？”
8. 若回答“需要”“添加”“还要”等，则继续询问标签；若回答“不需要”“不用”“不要”“开始检测”“确认”等，则打开相机、连接机械臂并开始检测。
9. 任务运行中听到“小智”后，暂停检测硬件使用和分拣执行：检测线程不关闭，`CameraSkill`、`SerialSkill`、`YoloSkill` 对象保留；相机采集连接和串口连接会临时断开，YOLO 不继续推理，也不会播报目标缺陷或向下位机发送 `N`/`D`。
10. 暂停后播报“我在。请问是要更换参数还是退出程序？”；听到“更换”后，重新执行模型和标签选择流程。
11. 更换完参数后，原检测线程直接更新参数并恢复运行；如果模型没有变化则复用已加载 YOLO 模型，只重新连接相机和串口；如果模型变化则只切换 RKNN 模型，不重建整个任务流程。
12. 听到“退出”后，无论当前处于哪个问题或任务状态，都会关闭分拣执行并退出程序。
13. 文字补充输入中的 `rm 1`、`rm 2`、`rm 3` 会分别移除当前已选的污渍、划痕、缺角检测标签。
14. 文字补充输入可直接输入 `1`、`2`、`3`；当前问题是模型时表示模型编号，当前问题是缺陷标签时表示对应缺陷标签。

## 标签与机械臂指令

- 1污渍：YOLO 标签 `Stain`，检测到后向 `/dev/ttyS9` 发送 `N`
- 2划痕：YOLO 标签 `Indentation`，检测到后向 `/dev/ttyS9` 发送 `D`
- 3缺角：YOLO 标签 `Corner defect`，检测到后向 `/dev/ttyS9` 发送 `D`

如果同时检测到多个目标标签，优先级为：划痕、缺角高于污渍；因此同时有划痕或缺角时发送 `D`，只有污渍时发送 `N`。

## 语音识别准确率优化

当前默认策略偏向减少误识别和重复触发：

- 保留单字有效命令：`1`、`2`、`3`、`一`、`二`、`三`、`好`、`是`、`不`、`否` 不再被当作过短文本丢弃。
- 参考电梯语音助手的采集方式，默认使用 ELF2/RK3588 板载 `card1 rockchipnau8822` 声卡采集，`48000Hz` 输入后重采样到 SenseVoice 需要的 `16000Hz`；输出默认也走 `rockchipnau8822`，即 3.5mm 耳机孔。
- 如果现场声卡枚举变化，可用 `PICOCLAW_INPUT_DEVICE=<编号>` 或 `PICOCLAW_OUTPUT_DEVICE=<编号>` 强制指定；也可用 `PICOCLAW_AUDIO_CARD_NAME=rockchipnau8822`、`PICOCLAW_INPUT_CARD_NAME=rockchipnau8822`、`PICOCLAW_OUTPUT_CARD_NAME=rockchipnau8822` 按设备名匹配。
- 默认输入通道为 `best`，会优先取双声道中能量更高的一路；如现场噪声偏大，可设置 `PICOCLAW_INPUT_CHANNEL=mean` 改回双声道平均。
- 增加原始噪声门限 `PICOCLAW_ASR_RAW_GATE`，默认 `0.0006`，避免自动增益把底噪放大成文字。
- VAD 默认值参考电梯语音助手并针对“1/2/3”等短指令调短最小语音：`PICOCLAW_VAD_THRESHOLD=0.55`、`PICOCLAW_VAD_MIN_SILENCE=0.08`、`PICOCLAW_VAD_MIN_SPEECH=0.16`、`PICOCLAW_VAD_BUFFER_SECONDS=100`。
- 默认关闭 VAD fallback 解码，减少短碎片误识别；如发现说话被漏听，可设置 `PICOCLAW_ASR_VAD_FALLBACK=1`。
- 默认开启 SenseVoice ITN，让数字识别更规整；如现场发现关键词被规范化干扰，可设置 `PICOCLAW_ASR_ITN=0`。
- 增加播报回声过滤，识别到系统刚播过的提示句时会忽略，减少自问自答。
- 短时间内包含式重复文本会被抑制，例如“小智”后紧跟“小智小智”不会重复触发。
